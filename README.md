SENKU OS is a browser-based desktop environment that i built entirely with HTML, CSS, and JavaScript. A lightweight operating system running straight in your browser, featuring draggable windows, desktop apps, a taskbar and even an anime streaming integration!

---

## ✨ Features

### 🖥️ Desktop Environment
- Draggable application windows
- Custom taskbar
- Window open/close system
- Fullscreen support

### 🔬 Science Logs
- Built-in science notebook (Not editiable!)
- Dr. STONE–inspired content
- Experiment and invention logs

### 📺 Dr. STONE Streaming Hub
- Season selector
- Episode search & browser
- Embedded video player
- Support for manual streaming links

### ⌚ Live System Clock
- Real-time clock & date display
- Updates every second automatically

### 🎨 Custom UI
- Orbitron typography
- Science-inspired design language
- Animated hover effects
- Desktop icon system
- Responsive layout
